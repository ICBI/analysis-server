Rserve.jar is required to compile.  

1)	Download the latest client jar file (use the exceptions version) from:
http://stats.math.uni-augsburg.de/Rserve/down.shtml after agreeing to their licensing
terms and conditions. 
2) The latest file at this time is: JRclient-RE817.jar
3) Copy the downloaded file into the lib directory under the analysis server install directory
and rename it to Rserve.jar .
